Table Flip Sopel extension
--------------------------

This is a simple extension to the Sopel IRC bot.


Requirements
------------
Install sopel irc bot

Installation
------------
git clone http://github.com/hemna/table-flip-sopel
cp table-flip-sopel to ~/.sopel/modules
restart sopel
